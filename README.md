# mrt
[![Build Status](https://travis-ci.org/Woutifier/mrt.svg?branch=master)](https://travis-ci.org/Woutifier/mrt)

An MRT (RFC6396) file parser implemented in Rust, using Nom.

## Key Features

* Implemented in Rust
* Fairly fast

## Supported elements

* PEER_INDEX_TABLE
* RIB_IPV4_UNICAST
* RIB_IPV6_UNICAST
* BGP4MP_MESSAGE_AS4
